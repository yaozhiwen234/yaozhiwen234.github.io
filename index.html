@NotNull,@NotBlank和 @NotEmpty的区别- @NotEmpty 用在集合上面(不能注释枚举) 对象不能是 null 并且相关对象的 size 大于 0。 - @NotBlank用在String上面 String 不能是 null 且去除两端空白字符后的长度(trimmed length)大于 0。 - @NotNull用在所有类型上面 对象不能是 null, 但可以是空集(size = 0)。
